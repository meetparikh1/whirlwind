# WhirlwindEdits

WhirlwindEdits is a social media management and video editing brand focused on creating engaging digital content for businesses and creators. This project represents the website built to showcase the services, brand story, and portfolio of WhirlwindEdits.

## About the Project

The website highlights the services offered by WhirlwindEdits, including video editing for reels, YouTube videos, and social media content. It is designed to present the brand professionally and attract potential clients looking for high-quality content creation.

## Features

- Brand introduction and storytelling
- Service cards displaying offered services
- Scrolling image sections
- Alternating image and text sections
- Clean and responsive layout
- Portfolio-style presentation

## Technologies Used

- HTML
- CSS

## Purpose

The goal of this project is to create a simple and effective portfolio website for a content editing agency, allowing clients to understand the brand and its offerings.
